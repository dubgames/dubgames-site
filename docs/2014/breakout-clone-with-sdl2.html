Breakout Clone with SDL2

 13 January 2014    Gram  



Source available here:Â https://github.com/dubgames/DubBreakout

Tech: C, SDL2

Features:

  • Atari-style graphics and sounds
  • The velocity of the ball depends on where it hits the paddle
  • Speed increase for bricks closer to the top
  • Scoring and lives
  • Levels with increasing ball speed
  • Keyboard input
  • Cheat codes to have infinite lives and skip levels (for testing purposes)

Features that you could add:

  • Make it a closer match to the Atari original
  • Add joystick or mouse support (edit: Controller support added!)
  • Re-skin with improved graphics and sound
  • Multi-player support (edit: added!)
  • Add more animations and sound for level start/end, block destruction
  • Custom levels from disk

Possible projects you could make:

  • Use the source as the basis of a Pong clone
  • Or, an Arkanoid clone
  • Create your own game with a similar mechanic
